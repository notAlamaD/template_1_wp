# Global Bulletin — тема для новостных сайтов

Одна лаконичная WordPress-тема с hero-блоком и карточками для широких новостных изданий.

## Установка
1. Переименуйте папку с этой темой (если нужно) и поместите её целиком в `wp-content/themes/` вашего сайта.
2. В админке WordPress активируйте тему и при желании создайте меню «Primary Menu».
3. Добавьте миниатюры (featured images) к постам, чтобы карточки выглядели аккуратно.

## Особенности
- Поддержка `title-tag` и миниатюр записей.
- Hero-блок с последней публикацией и двухколоночные карточки для остальных новостей.
- Лёгкая адаптивная сетка и акцент на фотографии.
