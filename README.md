# WordPress news theme pack

Набор из трёх лёгких WordPress-тем под разные задачи новостных сайтов:

- **Global Bulletin** — для широких/международных редакций, с крупным героем и акцентом на фотографии.
- **Local Gazette** — компактный макет для гиперлокальных сообществ и районных изданий.
- **Tech Digest** — карточная сетка для технологических и стартап-изданий.

## Установка
1. Скопируйте нужную папку из `themes/` в каталог `wp-content/themes/` вашего сайта.
2. В админке WordPress активируйте тему и добавьте меню «Primary Menu» при необходимости.
3. Установите миниатюры (featured images) для постов, чтобы карточки смотрелись аккуратно.

## Быстрый обзор
- Все темы поддерживают `title-tag` и миниатюры записей.
- **Global Bulletin:** hero-блок с последней публикацией и двухколоночные карточки для лент.
- **Local Gazette:** сетка карточек с лёгким теневым стилем и полосой-слоганом «Новости рядом с вами».
- **Tech Digest:** тёмная карточная сетка, «Spotlight» на отмеченную категорию `featured`, липкий header.

При желании обновите цвета и шрифты в `style.css` каждой темы под бренд издания.
