# Global Bulletin — тема для новостных сайтов

Одна лаконичная WordPress-тема с hero-блоком и карточками для широких новостных изданий. В репозитории нет обёртки `themes` — все файлы темы находятся прямо в корне.

## Установка
1. Создайте папку будущей темы в `wp-content/themes/` (например, `global-bulletin`) и скопируйте в неё все файлы из корня репозитория.
2. Убедитесь, что старой папки `themes/` в структуре проекта не осталось — она больше не используется и добавлена в `.gitignore`, чтобы не вернуть её случайно.
3. В админке WordPress активируйте тему и при желании создайте меню «Primary Menu».
4. Добавьте миниатюры (featured images) к постам, чтобы карточки выглядели аккуратно.

## Особенности
- Поддержка `title-tag` и миниатюр записей.
- Hero-блок с последней публикацией и двухколоночные карточки для остальных новостей.
- Лёгкая адаптивная сетка и акцент на фотографии.
